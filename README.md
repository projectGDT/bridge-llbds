# llplguin

(正式名称: **GDT-Bridge-LL**)

projectGDT 的子项目之一。

服务端插件的 LiteLoaderBDS 实现，与中心服务器通讯，进行向云端请求白名单等操作。

插件是完全开源的。为确保安全，仅保留了有限的操作玩家数据的接口。远程执行命令的功能将不予实现。
